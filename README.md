🧾 Inventory Management System

A web-based Inventory Management System built using Django to manage products, track stock, and monitor transactions efficiently.

---

🚀 Features

- ✅ Product Management (CRUD operations)
- 📦 Stock IN/OUT tracking
- 📊 Transaction History
- 🔍 Search functionality
- 🔐 User Authentication
- ⚠️ Low Stock Alerts
- 🎨 Responsive UI using Bootstrap

---

🛠️ Tech Stack

- Python
- Django
- SQLite
- Bootstrap
